# Proyecto-2-Resumen-Ejecutivo-de-Ventas-Walmart
Transformar datos Walmart en un resumen ejecutivo que oriente decisiones de presupuesto e inventario. Mediante KPIs clave (ventas por m² y participación por departamento) y dashboards interactivos, identificamos departamentos eficientes y áreas con bajo potencial. Comunicamos hallazgos con método C-F-I para impacto directo en dirección comercial.
